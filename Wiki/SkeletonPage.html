> Page History
vPage History

This section details some neat history about this page and the Wiki.


Development

This page was built in simultaneity with the Wiki's layout, being where different designs were tested. As such, it includes at least one instance of several features that needed to be tested. Once design and testing were complete, much of the old placeholder text was replaced with what you're reading now. Notably, this was the only Jabiveir Wiki page built from scratch.


Proceeding Use

The development of this page was for more than just layout design and bug testing, as the source code was used as a template for every other Jabiveir Wiki page made afterwards (thus, every other page). This was done by taking the code as a string, identifying which parts would vary between pages, and making variables for those sections; a program was then made to input pre-written text for those variables and splice them into the code, thereby easily making a complete Wiki page. The program that was used is being written about vaguely because, as of writing this, I haven't made it yet.


The Rest of the Page

There used to be more sections to this page, but they have since been removed, as they were just copy-pasted nonsense meant to test the page's scrolling.
